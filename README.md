# Microprocessor-Course-Project
The Project Requirements: 

Design an 8085 microprocessor system with the following specifications
Two 8K  EPROM chip.
Two 4K  RAM chips.
One peripheral-mapped Seven-Segment LED displays with address 35H.
One memory-mapped input port (DIP switches).
Memory Foldback is not allowed
All necessary circuits should be included (reset circuit, clock generation, bus demultiplexing, latches, decoders...)
